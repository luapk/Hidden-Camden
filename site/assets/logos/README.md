# Partner logo assets

Drop the official partner logo files here and they replace the typographic
placeholders automatically. No code change needed.

Each logo tile on the site loads an image; if the file is missing it falls
back to a styled wordmark. So the site works today, and upgrades the moment
you add the real asset.

## Expected filenames

| File | Where it shows | Notes |
|------|----------------|-------|
| `dr-martens.svg` | Sponsors — headline partner | Prefer white/light version (sits on dark) |
| `guinness.svg` | Sponsors — drinks row | White/light version |
| `tanqueray.svg` | Sponsors — drinks row | White/light version |
| `diageo.svg` | Sponsors — drinks row | White/light version |
| `roundhouse.svg` | Community — charity tile | Dark version (sits on paper) |

## Rules of thumb

- SVG preferred, transparent background. PNG at 3x also fine — change the
  extension in `index.html` if so.
- Logos on the dark sponsor wall want a white or light-monochrome cut.
- The Roundhouse tile is on a paper background, so use the dark logo there.
- Keep artwork within the trademark holder's brand guidelines. These are
  confirmed partners; use the assets they supply.
